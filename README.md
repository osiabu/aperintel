# Aperintel

Aperintel is an intelligence infrastructure company building AI-native systems that structure and enhance intelligence across organisations, communities, and individuals.

This repository contains the official Aperintel website, hosted via GitHub Pages.

## Live Site

https://yourusername.github.io/aperintel

## Structure

Single-file static site. No build process required.

- index.html — complete site (all pages, styles, and scripts)

## Pages

- Home
- About
- Platform
- Products
- Contact / Request Access

## Deployment

1. Clone or fork this repository
2. Replace index.html with your updated file
3. Go to Settings → Pages → Source: main branch, root folder
4. Site will be live at your GitHub Pages URL

## Stack

HTML · CSS · Vanilla JavaScript

Fonts loaded via Google Fonts (Syne, DM Sans).
No frameworks. No dependencies. No build step.
